# FlyPi_LarvalChemotaxis

Original FlyPi developed by https://github.com/amchagas/Flypi
Modified by Ajinkya Deogade (ajideo05_at_gmail.com) and Daeyeon Kim (daeyeon.kim_at_crg.eu)
